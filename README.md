## wtf-gulp-starter

**po sklonowaniu repozytorium zainstaluj Node ze strony [NodeJS](https://nodejs.org/) i**

`npm install -g gulp-cli`

`npm install`

`gulp`

Żeby opublikować swoją pracę na gh-pages uruchom `npm run deploy`
